# JS/TS Mastery — 300 Questions

A personal learning journal working through 300 JavaScript, TypeScript, and DSA concepts.
Updated daily. No fluff — just clean code and clear explanations per file.

## What this is

This is not a course or a library. It's a structured self-study repo where each file
is one concept, explained and demonstrated with runnable code.

Goal: solid understanding from JS fundamentals → advanced TypeScript → DSA → System Design.

## Requirements

| Tool      | Version  |
|-----------|----------|
| Node.js   | ≥ 18.0.0 |
| npm       | ≥ 8.0.0  |

Install via nvm (recommended):
```bash
nvm install 20 && nvm use 20
```

## Setup

```bash
git clone https://github.com/seema24jauhari/js-ts-mastery.git
cd js-ts-mastery
npm install
```

## Running a solution

```bash
# JavaScript
node solutions/js/js-fundamentals/Q001-equality-coercion.js

# TypeScript (coming as topics progress)
npx ts-node solutions/ts/Q197-ts-basics.ts
```
