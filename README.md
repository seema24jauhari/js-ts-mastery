# JS/TS Mastery — 300 Questions

A structured curriculum covering JavaScript, TypeScript, and DSA.

## Requirements

| Tool | Minimum Version | Why |
|---|---|---|
| **Node.js** | ≥ 18.0.0 | ES2022 features, native `structuredClone`, `Array.at()` etc. |
| **npm** | ≥ 8.0.0 | Workspaces support (ships with Node 16+) |
| **Git** | any | Version control |

> **Recommended:** Node 20 LTS (Active LTS as of 2024). Install via [nvm](https://github.com/nvm-sh/nvm):
> ```bash
> nvm install 20
> nvm use 20
> ```

---

## Setup

```bash
# 1. Clone the repo
git clone https://github.com/seema24jauhari/js-ts-mastery.git
cd js-ts-mastery

# 2. Install all dependencies
npm install

```
